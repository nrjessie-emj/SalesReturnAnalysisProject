# Product Return Rate Analysis

## Overview
This project analyzes product-level data to understand factors that influence return rates, including price, discount rate, category, and other product attributes.

## Objective
The goal of this analysis is to identify patterns in return behavior and provide insights that could help improve product performance and reduce return rates.

## Tools Used
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

## Key Insights
- Return rates are not strongly explained by any single variable  
- Higher-priced products show slightly higher return variability  
- Discount rate has a non-linear relationship with returns  
- Grouped analysis reveals more structure than raw correlations  

## Files in this Repository
- analysis.ipynb → full analysis notebook
